# Knowledge: Deno KV Tips

## 1. キー設計
- prefix を使ったリスト取得が可能。
- 例: `["users", discordUserId]`

## 2. 安全な運用
- ブラウザや外部から直接 KV を操作するエンドポイントは、本番環境ではコメントアウトまたは認証を必須にする。
- `Deno.cron` を使った定時実行との組み合わせで、データのクリーンアップなどが可能。

## 3. 制限事項
- Deno Deploy の Free Tier では KV の読み書き回数に上限があるため、頻繁なポーリングなどは避ける。
- `expireIn` オプションは Deno 1.32.0 以降で使用可能。
